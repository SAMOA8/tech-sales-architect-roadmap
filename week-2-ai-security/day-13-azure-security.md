# Day 13 — Azure Security & AI Services

## Microsoft Entra ID (formerly Azure AD)
- Identity platform: SSO, MFA, Conditional Access
- B2B (partners) | B2C (customers)
- Compare to AWS: Entra ID = IAM Identity Center + Cognito

## Microsoft Defender for Cloud
- CSPM + CWPP (Cloud Workload Protection Platform)
- Security score, recommendations, compliance dashboard
- Compare to AWS: Security Hub + GuardDuty

## Microsoft Sentinel
- Cloud-native SIEM + SOAR
- Collects logs from Azure, AWS, GCP, on-prem
- AI threat detection + automated response playbooks
- Compare to AWS: more powerful as a dedicated SIEM than AWS native

## Azure Key Vault
- Secrets/keys/certificates — like KMS + Secrets Manager combined

## Azure DDoS Protection / Azure Firewall
- Network protection, similar to Shield + Network Firewall

## Azure AI Services
- **Azure OpenAI Service** — GPT-4, GPT-4o, DALL-E (locked to OpenAI models)
- **Microsoft Copilot** — M365, GitHub, Security
- **Azure AI Studio** — like SageMaker Studio
- **Azure Cognitive Services** — pre-built AI APIs
