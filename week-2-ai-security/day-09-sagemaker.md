# Day 9 — Amazon SageMaker

## Core Components
- **Studio** — IDE for ML
- **Data Wrangler** — visual data prep
- **Feature Store** — centralized features
- **Training Jobs** — managed distributed training
- **Hyperparameter Tuning** — auto-optimize
- **Experiments** — track training runs
- **Model Registry** — version models
- **Endpoints** — real-time inference
- **Batch Transform** — bulk inference
- **Pipelines** — MLOps automation
- **Canvas** — no-code ML for analysts
- **Autopilot** — AutoML

## SageMaker Security
- VPC isolation for training jobs
- KMS encryption (data at rest, in transit)
- IAM granular permissions
- Network isolation (no internet for training containers)
- **Model Monitor** — detect data/model drift in production
- **Clarify** — bias detection, model explainability (SHAP)
- **Model Cards** — document model use, risks, performance
