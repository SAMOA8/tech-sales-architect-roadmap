# Day 10 — Amazon Bedrock & Foundation Models

## What Bedrock Is
- Fully managed access to foundation models via API
- No infrastructure to manage
- Pay per token (input + output)
- **Critical:** Customer data is NOT used to train models — guaranteed contractually

## Available Models on Bedrock
- **Anthropic Claude** (3.5 Sonnet, Opus, Haiku) — best reasoning, analysis, coding
- **Meta Llama** — open-source, fine-tuneable
- **Amazon Titan** — AWS native (text + embeddings)
- **Mistral** — efficient European models
- **Stability AI** — image generation
- **Cohere** — enterprise search/RAG

## Bedrock Features
- **Agents** — AI systems that take multi-step actions
- **Knowledge Bases** — RAG with your company docs
- **Guardrails** — content filtering, PII redaction, topic blocking
- **Model Evaluation** — compare models side-by-side
- **Fine-tuning** — adapt models to your domain

## Amazon Q
- **Q Business** — enterprise AI assistant connected to your data
- **Q Developer** — coding assistant (vs GitHub Copilot)
- **Q in AWS Console** — answers questions about your AWS environment
