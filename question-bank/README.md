# Question Bank — 150 Interview Q&A

A curated collection of 150 interview-style questions and detailed answers for Tech Sales Architect roles focused on AWS Cloud Security and AI Security.

## How to Use

- Read 10 questions per day
- Answer each one **out loud** before reading the answer
- Mark questions where you struggled — revisit them weekly
- By Day 30, you should be able to answer all 150 confidently
