# Certification Roadmap

## Priority Order for a Tech Sales Architect

### 1. AWS Certified Security – Specialty (Priority: Highest)
- **Why:** Directly relevant to your target role
- **Format:** 65 questions, 170 minutes
- **Cost:** $300 USD
- **Prerequisites:** None (recommended: 5 years security experience)
- **Difficulty:** Hard
- **Prep time:** 4–6 weeks if you have AWS experience
- **Resources:**
  - AWS Skill Builder (free) — Exam Prep: AWS Certified Security Specialty
  - Adrian Cantrill course (paid, very comprehensive)
  - Stéphane Maarek course on Udemy (paid)
  - Whizlabs / TutorialsDojo practice exams (paid)

### 2. AWS Certified Solutions Architect – Associate (Priority: High)
- **Why:** Most recognized AWS cert; expected baseline
- **Format:** 65 questions, 130 minutes
- **Cost:** $150 USD
- **Difficulty:** Medium
- **Prep time:** 4 weeks
- **Resources:**
  - AWS Skill Builder
  - Adrian Cantrill / Stéphane Maarek courses
  - TutorialsDojo practice exams

### 3. AWS Certified AI Practitioner (Priority: High)
- **Why:** New, growing demand, your differentiator
- **Format:** 85 questions, 120 minutes
- **Cost:** $100 USD
- **Difficulty:** Foundational
- **Prep time:** 1–2 weeks
- **Resources:**
  - AWS Skill Builder (free official prep)

### 4. CompTIA Security+ (Priority: Medium)
- **Why:** Vendor-neutral baseline, often required by US government roles
- **Cost:** $392 USD
- **Difficulty:** Medium
- **Validity:** 3 years

### 5. Microsoft AZ-900 (Priority: Medium)
- **Why:** Shows multi-cloud awareness for Azure-friendly customers
- **Format:** 40-60 questions, 60 minutes
- **Cost:** $99 USD
- **Difficulty:** Foundational

### 6. Google Cloud Digital Leader (Priority: Low)
- **Why:** Shows multi-cloud awareness for GCP
- **Cost:** $99 USD
- **Difficulty:** Foundational

### 7. Certified Cloud Security Professional (CCSP) (Priority: Future)
- **Why:** Premium vendor-neutral credential — high value but requires experience
- **Cost:** $599 USD
- **Prerequisites:** 5 years IT experience including 3 years infosec, 1 year cloud
- **Best for:** mid-career, after Security Specialty

## My Plan

- **Month 2:** Pass AWS Solutions Architect Associate
- **Month 3:** Pass AWS AI Practitioner
- **Month 4–6:** Add Azure AZ-900 + Google CDL
