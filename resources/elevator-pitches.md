# Elevator Pitches & Practice Scripts

## Personal Pitch — 30 Seconds
> "I'm a Tech Sales Architect specializing in AWS cloud security and AI security services. I help companies secure their cloud, meet compliance requirements, and safely adopt AI. I've built a security audit tool on GitHub that I run against real AWS accounts — so I can speak to both engineering teams and CISOs with credibility."

## Personal Pitch — 60 Seconds (Networking)
> "I came into cloud security through hands-on AWS Security training and immediately built a portfolio project — a Python tool that audits AWS accounts for misconfigurations. When I ran it against my own account, it found 7 real security issues, including 17 security groups exposing SSH to the internet. That experience taught me what most companies face daily — and how AWS native services like GuardDuty, Security Hub, and Config solve those problems. I'm focused on helping organizations adopt cloud and AI safely, especially with services like Amazon Bedrock that need new security thinking. I'm looking for a Tech Sales Architect role where I can bridge that gap between technical capability and business outcomes."

## Customer-Facing Pitches

### AWS Security to a CEO
> "Imagine your business moved from a locked office building to a shared skyscraper. Amazon built the building and maintains the locks on the front door — but you're responsible for locking your own office, controlling who has keys, and protecting what's in your filing cabinets. AWS security services act like a 24/7 security guard, a vault for your most sensitive documents, and a complete camera system that records everything. So when something goes wrong, we know exactly what happened and who did it."

### AWS Security to a CFO
> "Cloud security is one of the highest-ROI investments you can make. The average data breach costs $4.88 million. GuardDuty costs $500 a month and can detect intrusions in minutes — versus the industry average of 194 days. If GuardDuty prevents even one incident in 3 years, the ROI is 800x. Beyond risk avoidance, security certifications like SOC 2 unlock enterprise customers who otherwise can't buy from you."

### AWS Security to a CISO
> "AWS gives you three things most organizations struggle to build internally: visibility, automation, and scale. Visibility through Security Hub aggregating 10+ tools into one dashboard. Automation through GuardDuty's ML-based threat detection that finds what rules miss. Scale through serverless event-driven response — your tier-1 SOC work happens in EventBridge and Lambda, freeing your team for real investigation. And it deploys in minutes, not months."

### Bedrock to a Skeptical Customer
> "I understand the AI security concern — most companies worry their data will train someone else's model. Bedrock is built differently. Your data stays in your AWS account, your AWS region, never used to train models. You get IAM-based access control, VPC private connectivity, content guardrails, PII redaction, and full audit logging. And you choose from multiple model providers — Anthropic, Meta, Amazon, Mistral, Cohere — so you're never locked in. Your CISO can audit it. Your legal team can review it. Your data scientists can build with it. Same AWS security model, applied to AI."
