# Day 28 — Mock Technical Interviews

## Technical Questions to Answer Aloud
1. Design a secure architecture for a 3-tier web application on AWS.
2. A customer's S3 bucket was made public. Walk through incident response.
3. How would you architect a HIPAA-compliant system on AWS?
4. A customer wants to deploy AI chatbot for customer service. What security controls?
5. Explain the difference between GuardDuty and Security Hub to a 10-year-old.

## Whiteboard Challenges
- Draw a Zero Trust architecture using AWS services
- Draw a multi-account AWS security architecture (Organizations + OUs + SCPs + Security Hub)
- Draw a secure Bedrock chatbot architecture (VPC + Guardrails + IAM + logging)

## Behavioral Questions (STAR format)
1. Tell me about a time you had to explain a complex technical concept to a non-technical person.
2. Tell me about a time you disagreed with a customer.
3. Tell me about a time you didn't know an answer to a customer's question.
4. Tell me about a time you missed a deadline.
5. Why this company specifically?
