# Day 22 — AWS Security Specialty Prep (Part 1)

## Exam Overview
- 65 questions, 170 minutes
- Domains:
  1. Threat Detection and Incident Response (14%)
  2. Security Logging and Monitoring (18%)
  3. Infrastructure Security (20%)
  4. Identity and Access Management (16%)
  5. Data Protection (18%)
  6. Management and Security Governance (14%)

## Domain 1 — Threat Detection & Incident Response
- GuardDuty findings and remediation workflow
- Security Hub: finding → notification → remediation
- Incident response: isolate (SG) → investigate (CloudTrail) → remediate (Lambda)
- AWS Abuse team notifications
- EventBridge rules for automated response

## Domain 2 — Security Logging & Monitoring
- CloudTrail: management vs data events, log file validation, multi-region
- CloudWatch: log metric filters, alarms, EventBridge
- VPC Flow Logs: fields, ACCEPT vs REJECT
- S3 access logging vs CloudTrail data events
- Comparing log destinations: CloudWatch vs S3 vs Kinesis Firehose
