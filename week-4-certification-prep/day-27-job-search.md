# Day 27 — Job Search Strategy

## Target Job Titles
- Sales Engineer (most common)
- Solutions Architect (pre-sales)
- Cloud Security Architect
- Technical Account Manager (AWS TAM)
- Pre-Sales Security Consultant
- Cloud Solutions Consultant

## Target Companies

### AWS itself
- TAM, Solutions Architect, Specialist Solutions Architect (Security)

### AWS Partners (consulting/SI)
- Accenture, Deloitte, PwC, Capgemini, Wipro, Slalom

### Security Vendors
- CrowdStrike, Palo Alto Networks, Wiz, Lacework, Splunk

### MSSPs
- Managed security service providers

### Startups
- AI security: Protect AI, HiddenLayer, Robust Intelligence, Lakera

## Resume Optimization

### Structure
1. **Summary** — 3 sentences: who, what you do, differentiator
2. **Technical Skills** — AWS services, Python, security frameworks
3. **Projects** — link to GitHub (audit tool, roadmap repo)
4. **Certifications** — in progress + completed
5. **Experience** — STAR format

### ATS Keywords to Include
Cloud Security, AWS, IAM, GuardDuty, Zero Trust, CSPM, SIEM, Compliance, Python, boto3, Bedrock, AI Security, Solutions Architecture, Pre-Sales, ROI, TCO, MEDDIC

## Networking Communities
- AWS re:Post (official community)
- Cloud Security Alliance (CSA)
- ISACA
- LinkedIn groups: "AWS Users Group", "Cloud Security Professionals"
- Attend: AWS meetups, BSides, virtual conferences
